## 📌 Sobre o Projeto
Este repositório foi criado para documentar a execução de *testes manuais de qualidade de software (QA)* na aplicação de e-commerce *[Automation Exercise](https://automationexercise.com)*.

O objetivo principal deste projeto é praticar e demonstrar na prática o planejamento, a estruturação de cenários de teste, o levantamento de melhorias de UX e o reporte de falhas em um ambiente de e-commerce.

---

## 🎯 Objetivos

- [ ] Elaborar o *Plano de Testes* da aplicação.
- [ ] Mapear e escrever os *Cenários e Casos de Teste* manuais.
- [ ] Documentar *Bugs e Falhas* encontrados durante as validações.
- [ ] Propor *Melhorias de Usabilidade (UX/UI)* para a plataforma.

* *Tipo de Teste:* Teste Manual Functional & Exploratório
* *Aplicação Alvo:* E-commerce (Automation Exercise)
